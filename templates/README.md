A high-end Flask application for managing patient flows using advanced data structures.

## 🚀 Project Overview
This system provides a seamless experience for clinic staff to register patients, manage a waiting line, and undo actions if a mistake is made.

## 🛠 Technical Features (Grading Criteria)
* **OOP:** Implemented `Patient` and `ClinicQueue` classes.
* **FIFO Queue:** Manages the main patient list.
* **LIFO Stack:** Powers the "Undo Call" functionality.
* **Standard API:** Uses the `datetime` module for precise arrival tracking.
* **Web UI:** A minimalist, premium black-and-white interface.

## 💻 How to Run
1. Install Flask: `pip install flask`
2. Run the app: `python app.py`
3. Visit: `http://127.0.0.1:5000`
